Presenting my first participation in a Code/Game Jam

The Global Temperature Forecast is an interactive program that displays the earth’s yearly difference in temperature from 1880 to 2077. Of course, future dates are composed of 3 different prediction models.

  1. The First model is the Current Policies Forecast: Which is the increase if nothing changes in terms of habit from today
  2. The Second model is the Pledge Policies Forecast: This is what the government agreed on policies to implement to limit the rise of global temperature
  3. The Last model is Optimistic Forecast: This is the model that we as humans have to aim in the hope to minimize climate change.

Technical Details

The program was written in Lua using the Love2d Framework. Love is easy to learn a framework, similar to C++/ SFML. If you are a begineer or want to prototype quickly app or games check out love2d: Love2d website

Reflection

As my first published work, this entry is far from perfect. One major issue is the number of draw calls for drawing the graph. Each point is a separate draw call, which is fine on a desktop computer. However, the web build suffers from a slowdown. The love builds as a bug that is not present in the binaries.
