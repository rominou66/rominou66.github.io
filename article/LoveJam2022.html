Presenting my submission for the Love Jam 2022

Seismic Afterburner is a vertical shoot-em-up, based on the jam theme “Earthquake”. This game features a lot of screen shakes. The goal is simple, defeat the enemy planes protecting the enemy bomber.

Technical Details

As required by the game jam the game was written in Lua using the Love2d Framework. Check out the amazing framework: Love2d website

Reflection

This second participation in a game jam using love, my knowledge of the framework allows me to focus on understanding and implementing collision, Lua table for bullet and enemies and game states. After four entire days of coding, I achieve a result that I considered complete. Most importantly, the game is free of bugs. The organized codebase(with some cleaning needed) can be expanded further(more enemy, stage, weapons and bosses). In the future, I want to revisit this project to add more effect with shaders once I learn shading programming.
